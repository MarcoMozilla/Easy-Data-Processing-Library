# Lazy-Python
An simple python local data base

# How to install

```
pip install lzpy
```

# How to use

```
from lzpy import Table
Table.encoding = "utf-8" # choose encoding format
t=Table.read("filename") # if filename is "data.csv" write "data" inside parentheses
```
